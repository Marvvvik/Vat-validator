1. Открыть проект 
2. Создать базу данных через команду php bin/console doctrine:database:create или в ручную в локальной базе дынных 
3. сделать миграцию 

php bin/console make:migration
php bin/console doctrine:migrations:migrate

4. Создать пользователя командой php bin/console app:create-user test@getic.com admin
5. Запустить проект
